# Jade Collections

Premium ethnic wear e-commerce website.

## Setup

1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the development server.
3. Run `npm run build` to create a production build.

## Deploy to Vercel

1. Push your code to GitHub.
2. Go to [vercel.com](https://vercel.com) and sign in.
3. Click **Add New Project** and import your repository.
4. Vercel auto-detects Vite and the build settings.
5. Click **Deploy**.

The site will be live at `https://your-project.vercel.app`. Client-side routing (e.g. `/shop`, `/about`) is handled automatically via `vercel.json`.
